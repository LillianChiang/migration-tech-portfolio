# Skilled Migration Dashboard

## Overview

This project demonstrates a Power BI dashboard designed to analyse Australian skilled migration pathways.

The dashboard provides insights into:

- Skilled visa categories
- State nomination programs
- Occupation trends
- Invitation score trends
- Migration pathway comparisons

## Objectives

- Visualise skilled migration data
- Identify migration trends
- Support migration planning and decision-making
- Demonstrate Business Analysis and Data Analytics skills

## Planned Features

### Visa Categories

- Skilled Independent Visa (189)
- Skilled Nominated Visa (190)
- Skilled Work Regional Visa (491)
- Permanent Residence (Skilled Regional) Visa (191)

### Dashboard Visualisations

- Visa category distribution
- State nomination comparison
- Occupation demand analysis
- Invitation score trends
- Application pathway mapping

## Technologies

- Power BI
- Excel
- SQL
- Data Modelling
- Business Analysis

## Future Enhancements
